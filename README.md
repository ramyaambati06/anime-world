# Anime World

A responsive anime browsing web application built using HTML, CSS, and JavaScript.

## Features

* Anime search functionality
* Responsive UI
* Interactive anime cards
* Clean frontend design

## Tech Stack

* HTML5
* CSS3
* JavaScript
